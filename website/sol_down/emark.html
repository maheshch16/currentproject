E-marketing

Gate Access SW Platform of Coresonant: PortShield​

  • Integrated web based GPS platform for tracking and allotting vehicles.
  • Revenue generation through issue of permits (per use of man, vehicle, equipment)​.
  • Mobile payment gateway to customer.
  • Calls, messages and notifications methodology​.

Features:

  • Innovative software solution to optimize the allocation of transport for carrying of cargo(empty vehicles and TTs).​
  • Visibility of all the available transport to PTP office​.
  • Visibility of authorized transport to stakeholders.
  • Facility to requisition this transport dynamically​.
  • Intelligent / Smart scheduling​.
  • Capability to track the consignment till destination.
  • Payment gateway for transactions​.

Functinality:

  • Databank of the CHA, transporter agent, transporters​.
  • Approx rate fixation for the trips distance wise, destination wise, tonnage wise and type of transport​.
  • Automatic population of the list based on transport coming to port as per the permit requests( API Call)​.
  • Ride end trigger. Both for GPS as well as app.
  • Available transport and visibility to stakeholders​.
  • Demand can be placed by PTP, Stevedore, CHA, transporter agent etc​.
  • Requisition the transport based on charges (e market rules)​.
  • Additional charges based on features selected(Insurance, fuel/food)​.
  • Advance payment made and msg passed back to permit office for destination​.

Web Based Interface:

  • Interactive and state of the art solution​​.
  • Part module is deployed in seven seaports in India​.
  • Optimized routes and geo fencing​​.
  • Cloud based solution​.
  • Roles and logins​​.
  • Data security​.

Mobile App Interface:

  • Customized development​​​.
  • Features of aiscipro.​
  • Online booking/bidding system​.
  • Driver will be able to see route to be taken​​​.
  • Alerts and notifications can be pushed​​.
  • Empty vehicles can be seen and requisitioned like Ola/Uber​​​.
  • Once engaged, trip can be visible to all stakeholders​​.
  • Suitable alerts if diversion from a scheduled route​
